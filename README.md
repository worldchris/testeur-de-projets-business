# Testeur de Viabilité Business - Outil pour Éditeurs Web

Un widget interactif open-source conçu pour aider les webmestres à augmenter la rétention de leurs visiteurs tout en générant des revenus d'affiliation passifs, sans friction technique.

👉 **[Accéder à la page du projet et aux instructions d'intégration](https://worldchris.github.io/testeur-de-projets-business/)**

## 🎯 Pourquoi utiliser cet outil ?

Dans un écosystème web saturé, engager ses lecteurs est devenu indispensable. Ce widget permet à vos visiteurs de tester la viabilité de leur idée de projet web ou business en 3 questions rapides. 

Il a été pensé pour offrir un double avantage immédiat aux éditeurs de sites :
- **Rétention et SEO :** Transforme la lecture passive en interaction active, augmentant le temps passé sur la page.
- **Monétisation automatique :** Intègre un système d'affiliation personnalisable (notamment pour Systeme.io) pour rémunérer le trafic que vous générez, tout en apportant une vraie solution à vos lecteurs.

## ⚙️ Comment l'installer sur votre site ?

Pour garantir que le code d'intégration (iframe) reste propre et pour éviter les problèmes de formatage fréquents sur GitHub, l'intégralité du code prêt à copier-coller est hébergée sur notre page officielle.
